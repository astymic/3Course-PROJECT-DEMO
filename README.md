# LiLu — Взуттєва фабрика

> Веб-платформа для продажу взуття власного виробництва

**🌐 Живий сайт:** [https://3-course-project-demo.vercel.app](https://3-course-project-demo.vercel.app)
**🔧 Адмін-панель:** [https://3-course-project-demo.vercel.app/admin](https://3-course-project-demo.vercel.app/admin)

---

## Про проєкт

LiLu — повноцінний інтернет-магазин взуття, розроблений за три спринти:

| Sprint | Що зроблено |
|--------|-------------|
| Sprint 1 | Каталог, фільтри, кошик, оформлення замовлень, адмін-панель |
| Sprint 2 | Авторизація, особистий кабінет, live чат-підтримка |
| Sprint 3 | PostgreSQL, деплой на Vercel, мобільна адаптація |

---

## Функціонал

### Для покупця
- 🛍 Каталог з фільтрацією (категорія, колір, розмір, сезон, пошук)
- 🛒 Постійний кошик (зберігається між сесіями)
- 📦 Оформлення замовлення: Нова Пошта або кур'єр
- 👤 Особистий кабінет: замовлення, профіль, історія чату
- 💬 Чат-підтримка: FAQ-бот + живий оператор
- 🔗 Авторизація через телефон або email

### Для адміністратора
- 📋 CRUD товарів та управління залишками
- 📊 Управління замовленнями зі зміною статусів
- 👥 Управління користувачами та ролями
- 💬 Дашборд чату з відповідями в реальному часі

## Технічний стек

| | Технологія |
|--|------------|
| Frontend | Next.js 16, React 19, Tailwind CSS |
| Backend | Next.js API Routes |
| База даних | PostgreSQL (Supabase) |
| ORM | Prisma 5 |
| Хостинг | Vercel |
| Авторизація | HMAC-SHA256 + httpOnly cookie |

---

## Структура репозиторію

```
demo/
├── docs/            ← Технічна документація та звіти
├── sprint1/         ← Каталог, кошик, адмін (v1)
└── sprint2/         ← Повна версія (production)
```

> Активна версія — `sprint2/`

---

## Локальний запуск

```bash
cd sprint2
npm install
npx prisma migrate dev
node prisma/seed.js
npm run dev
```

Потрібен файл `.env` з `DATABASE_URL`, `DIRECT_URL`, `AUTH_SECRET`, `NP_API_KEY`.
